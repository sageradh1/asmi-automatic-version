Proper way to install Virtual Environment in ubuntu 18.04:
	1. apt-get update -y
	2. apt-get install -y python3-venv
	3. mkdir project_name
	4. cd project_name/
	5. python3 -m venv environment_name
	6. source ..../environment_name/bin/activate
	7. update essential packages with pip install -U setuptools pip
	8. install new packages : pip install termcolor
	8. deactivate

To Start a virtual environment with all the globle packages included:
	virtualenv venv_name --system-site-packages
To activate a virtual environment
	source ..../venv_name/bin/activate
To deactivate a virtual environment
	deactivate
