
Give permission
	sudo chown -v /home/ubuntu/workingDir/asmi-flask/
	sudo chown ubuntu:www-data /home/ubuntu/workingDir/asmi-flask/

To see what is wrong with 
	gunicorn run:app --error-logfile=- --access-logfile=-


To see error logs:
	sudo journalctl -u asmiflask
	Check the Nginx process logs by typing: sudo journalctl -u nginx
	Check the Nginx access logs by typing: sudo less /var/log/nginx/access.log
	Check the Nginx error logs by typing: sudo less /var/log/nginx/error.log
	Check the Gunicorn application logs by typing: sudo journalctl -u gunicorn
	Check the Gunicorn socket logs by typing: sudo journalctl -u gunicorn.socket
